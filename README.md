# Random Cat!

# Request

## Type A

### Step 1
1. goto **/cat**
2. then api will return **cat id**!

### Step 2
1. goto **/cat/{cat_id}**
2. then server will return **cat image** or **JSON**!

## Type B

### Step 1
1. goto **/meow**
2. then server will return **cat image** or **JSON**!


# Response

## JSON

```json
{
    "error": "reason of error"
}
```

## Image

- type will be **png**, **jpg** or **gif**.
- filename will be **cat._extension_**
